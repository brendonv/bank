#Simple Bank

## Setup

Prerequisites:
1. Node.js
2. npm
3. MongoDB

## Initialization 

1. `npm install`

## Usage

1. Start DB `mongod`

2. For stdin: `./index.js enter`
- Enter commands
- To exit: `ctrl-D`

3. `./index.js file [file]`

## Testing

- Make sure mongod process has been started 
1. `make test`
